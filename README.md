## Installation

First Fork this repo & then Clone this repo

```bash
  git clone https://github.com/<YOUR_USERNAME>/snapgram-mern-social-media-app.git
  cd snapgram-mern-social-media-app
```

Install the node_modules

```bash
  npm install
```

run the project

```bash
  npm run start
```
